# Nuestra Historia - Juan y Amgie

## Instrucciones rapidas

1. Coloca las fotos en /images con estos nombres exactos:
   - fondo.jpg
   - ano1.jpg, ano2.jpg, ano3.jpg, ano4.jpg, ano5.jpg, ano6.jpg
2. Coloca la cancion en /music/cancion.mp3
3. Sube TODA la carpeta a Netlify Drop: https://app.netlify.com/drop
4. Copia la URL generada y compartela.

## Personalizacion

Edita solo el archivo js/config.js para cambiar nombres, fecha, textos e imagenes.
